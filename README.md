Учебный проект яндекс практикума

Запуск dev сервера:
npm run dev

Сборка проекта:
npm run build

Сборка проекта и запуск EXPRESS:
npm run start

Проверка типов
npm run typecheck

Линтинг кода
npm run lint

Запуск тестов
npm run test
